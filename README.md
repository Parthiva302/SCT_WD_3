# Tic Tac Toe — Neon Dark

Vanilla HTML/CSS/JS tic-tac-toe web app with an unbeatable Minimax AI.

How to run:

1. Open [index.html](index.html) in a browser.
2. Use the mode buttons to switch between "Vs Computer" and "Vs Player".
3. Click cells to play. Restart resets the board.

Features:

- 3x3 responsive grid
- Player vs Computer (Minimax AI)
- Player vs Player
- Win/draw detection and highlighting
- Smooth neon dark UI
